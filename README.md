# 🧠 Memory Matching Game Pro

An interactive browser-based **Memory Matching Game** built using
**HTML, CSS, and JavaScript** featuring multiple difficulty levels,
timer countdown, sound effects, dark mode, animations, and high-score
tracking using localStorage.

🚀 Live Demo\
https://rutujapatil1299-maker.github.io/MEMORY_GAME

------------------------------------------------------------------------

## 🎮 Features

✅ Easy / Medium / Hard difficulty levels\
✅ Countdown timer system\
✅ Move counter tracking\
✅ High score storage using localStorage\
✅ Dark mode toggle 🌙\
✅ Background music & sound effects 🔊\
✅ 3D flip card animation\
✅ Victory popup screen with stats\
✅ Restart game functionality\
✅ Responsive layout design

------------------------------------------------------------------------

## 🛠 Technologies Used

-   HTML5
-   CSS3 (3D transforms & animations)
-   JavaScript (DOM manipulation & game logic)
-   Browser LocalStorage

## ▶ How to Run the Project

1.  Download or clone the repository
2.  Open the project folder
3.  Double-click **index.html**

Game will start in your browser 🎉

------------------------------------------------------------------------

## 🌐 Deploy Using GitHub Pages

Steps:

1.  Go to repository **Settings**
2.  Select **Pages**
3.  Choose **Deploy from branch**
4.  Select branch **main**
5.  Select folder **/ (root)**

Your project will be live at:

https://rutujapatil1299-maker.github.io/MEMORY_GAME
